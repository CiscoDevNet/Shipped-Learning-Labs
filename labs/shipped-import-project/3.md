# Shipped - Deploy your Build

1. Check the build logs.
 
    ![](posts/files/shipped-import-project/assets/images/7.PNG)

1. Click the **Deploy** tab, and click **New Environment** to create a new deployment environment.

    ![](posts/files/shipped-import-project/assets/images/8.PNG)

1. Enter a name and description for the new deployment environment, click **Save**.
 
    ![](posts/files/shipped-import-project/assets/images/9.PNG)

1. Click **Deploy Build**.

    ![](posts/files/shipped-import-project/assets/images/10.PNG)
